## 👋 关于我们

**NARC（Neepu Auto Robotic Collective）** 是一个由学生发起、面向学生的开源知识共享组织。

我们相信：**代码不该写完就丢在硬盘里吃灰。** 每一次课设、每一个实验、每一行调试通过的代码，都值得被记录和分享。无论是自动化、机器人相关专业课的课程设计，还是自己折腾的小项目，都可以在这里找到安身之处。

## 📦 这里有什么

- 📚 **专业课课设** — 自动化、机器人、嵌入式等方向的课程设计代码与报告
- 🧪 **实验代码** — 课内实验、开放实验的完整工程
- 🛠️ **实用工具** — 提高日常开发效率的小工具、脚本
- 🚀 **课外项目** — 竞赛、大创、兴趣驱动的个人/团队项目
- 📝 **复习资料** — 笔记、总结、历年考题整理（如果有的话 😄）

## 🤝 贡献你的代码

不管是课设、实验，还是复习笔记，只要你觉得对学弟学妹有用，都欢迎提交！

1. **Fork** 对应的仓库（或联系我们新建一个）
2. 把你的代码/资料整理好放进去
3. 写清楚 **README**（项目简介、运行环境、使用方法）
4. 发起 **Pull Request**

> 💡 上传前记得检查有没有不小心带上个人信息或敏感数据哦。

## ⚠️ 免责声明

本组织所有代码和资料仅供 **学习参考**。请遵守学校相关规定，**独立完成** 自己的课设和作业。直接照搬导致的一切后果（查重被抓、知识没学到、毕业设计翻车等）本组织概不负责 😅
